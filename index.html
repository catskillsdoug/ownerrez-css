Typography Audit

OwnerRez Reset Brand - Defined styles vs Bootstrap/OwnerRez inherited defaults

Consistency Checklist - Items Needing Attention

  • !
    Letter-spacing - Only explicitly set to 0 on headings. Buttons, labels, and body text inherit from Bootstrap (which may vary).
  • !
    Line-height - Not explicitly set on buttons or form controls. Bootstrap default is 1.428571429, but elements may vary.
  • !
    Link colors - Heading links are set to #000, but general links inherit Bootstrap blue (#337ab7).
  • !
    Focus states - No explicit focus styles defined. Bootstrap defaults may show blue outlines or box-shadows.
  • !
    Font family - Not explicitly set. Inherits from OwnerRez/Bootstrap which may use different stacks in different contexts.
  • !
    Transition effects - Not defined. Bootstrap may add transitions on hover/focus that feel inconsistent with Swiss design.

Recommended CSS Additions for Consistency

Add these rules to styles.css
/* ============================================
   TYPOGRAPHY CONSISTENCY FIXES
   ============================================ */

/* Global letter-spacing reset */
body,
.btn,
.form-control,
label {
  letter-spacing: 0 !important;
}

/* Global line-height for interactive elements */
.btn,
.btn-default,
.form-control,
.navbar-cabin a {
  line-height: 1.5 !important;
}

/* Override Bootstrap link colors */
a {
  color: #000000 !important;
}

a:hover,
a:focus {
  color: #000000 !important;
  text-decoration: underline !important;
}

/* Remove Bootstrap focus glow */
.btn:focus,
.btn-default:focus,
.form-control:focus {
  outline: none !important;
  box-shadow: none !important;
  border-color: #000000 !important;
}

/* Or use a Swiss-style focus indicator */
.btn:focus,
.form-control:focus {
  outline: 2px solid #000000 !important;
  outline-offset: 2px !important;
}

/* Remove any transitions for sharp interactions */
.btn,
.btn-default,
.form-control,
a {
  transition: none !important;
}
