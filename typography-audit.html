Design Principles Applied

Swiss Design

No rounded corners (border-radius: 0) applied globally to all elements including buttons, form controls, images, and cards.

Uppercase Treatment

Buttons, labels, form placeholders, and calendar headers use uppercase text-transform for a bold, authoritative appearance.

Heavy Weight

Font-weight: 700 used consistently for headings, buttons, form controls, and navigation elements.

Black Borders

Consistent 1px solid #000000 borders on interactive elements: buttons, form controls, sidebar container.

Observations & Notes

Font Family

The site uses the default Bootstrap/system font stack. No custom web fonts are specified in the custom CSS overrides. The underlying OwnerRez platform likely uses a standard sans-serif stack.

Limited Type Scale

Only two distinct sizes are defined: 30px for headings and 14px for everything else (body, buttons, forms). This creates a very flat hierarchy with strong contrast between headings and content.

Responsive Considerations

Padding-right adjustments for text blocks: 15% on desktop, 10% on tablet, 5% on mobile. No responsive font-size adjustments are specified.
